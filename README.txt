# Portafolio Web - Kevin Muñoz Garro
Mi portafolio personal como desarrollador web, construido con JavaScript moderno y CSS responsivo.

## Características
- Secciones: Home, About, Skills, Projects, Contact
- Diseño responsivo con CSS moderno
- Código modular en JavaScript (clases importadas en `app.js`)
- Paleta de colores personalizada con variables CSS
- Footer con enlaces sociales

## Tecnologías
- HTML5
- CSS3 (Flexbox, Grid, clamp() para tipografía responsiva)
- JavaScript (ES6+, modular)
